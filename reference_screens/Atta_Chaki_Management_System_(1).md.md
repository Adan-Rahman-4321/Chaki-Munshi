## **Atta Chaki Management System (Web Application)** 

## **Project Overview** 

A simple, lightweight, and user-friendly **web-based system** for managing daily operations of an Atta Chaki (Flour Mill). 

This system is designed to run in a browser and can be deployed for free online. 

## **Project Name Suggestions** 

- Chaki Munshi Web 

- My Atta Chaki Online • Flour Mill Manager 

## **Purpose** 

This web app helps Atta Chaki owners manage: 

- Wheat (Gandam) records 

- Cleaning (Safai) weight 

- Flour (Atta) distribution 

- Payments & Udhaar tracking 

- Daily, Monthly & Yearly reports 

## **Platform** 

- Web Application (Browser-based) 

- Works on Mobile + Desktop 

- Online (with optional offline support via caching) 

## **Target User** 

- Shopkeeper / Chaki Owner
